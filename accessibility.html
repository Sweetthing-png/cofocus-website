Accessibility Statement

CoFocus aims to make this website accessible according to WCAG 2.2 Level AA. This website is a mock webshop created for an academic exercise.


Accessibility Measures

The website was designed with accessibility in mind. The following measures were considered:

  • Clear page structure with semantic HTML elements such as header, nav, main, section and footer.
  • Logical heading order using h1, h2 and h3.
  • Alternative text for meaningful images, such as the CoFocus logo.
  • Decorative icons are hidden from screen readers where appropriate.
  • Readable text size and sufficient spacing between elements.
  • Keyboard accessible navigation links and buttons.
  • Responsive layout for different screen sizes.
  • No autoplaying audio or video content.
  • Consistent navigation and footer links across the website.
  • The website contains a decorative muted video used for atmosphere. The video does not autoplay with sound and includes an aria-label for screen reader support.

Testing

The website was reviewed using the WCAG 2.2 AA checklist provided by DigitalA11Y.
The website was also tested locally with AXE DevTools in Google Chrome. The automated check did not report accessibility issues at the time of testing.


Known Limitations

Some visual background images are decorative and are implemented through CSS. They do not contain important information and therefore do not require alternative text. As this is a mock website, some buttons and forms do not perform real transactions.
